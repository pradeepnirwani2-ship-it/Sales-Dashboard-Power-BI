Interactive Power BI Sales Dashboard for Retail Store Sales Analysis using Power BI, Excel, Power Query and DAX.

# Retail Sales Dashboard - Power BI

## Project Overview

This project is an interactive Power BI dashboard developed to analyze retail store sales performance. It helps business users monitor sales, customer purchasing patterns, product performance, payment methods, and monthly sales trends through interactive visualizations.

The dashboard enables users to filter data dynamically and make better business decisions using real-time insights.

##  Tools Used

- Power BI Desktop
- Microsoft Excel
- Power Query
- DAX
- Data Modeling

---

##  Dashboard Features

### KPI Cards
- Total Sales
- Total Units Sold
- Number of Bills
- Average Bill Value
- Large Bill Value
- Large Bill Count

### Visualizations
- Monthly Sales Trend
- Sales by Product
- Sales by Locality
- Sales by Payment Method
- Sales by Bill Size

### Interactive Filters
- Date Range
- Locality
- Payment Method

---

## Key Business Insights

- Track monthly sales performance.
- Identify top-selling products.
- Compare sales across different localities.
- Analyze customer payment preferences.
- Monitor average bill value.
- Understand large bill transactions.
